# OONT — Assignment

A repository for the OONT assignment project, complementing the [oont-api](https://github.com/VivekRahal/oont-api) backend service.

## About

This repository contains assignment-related work for the OONT (Grocery Inventory & Order) project. The corresponding API backend is built with NestJS, PostgreSQL, and Prisma — see the [oont-api repository](https://github.com/VivekRahal/oont-api) for the full backend implementation.

## Project Overview

The OONT system is a grocery inventory and order management platform that includes:

- **Product Management** — Browse and search grocery products by category
- **Cart System** — Add, update, and remove items from a shopping cart
- **Order Processing** — Place orders with atomic stock management and concurrency control
- **Inventory Tracking** — Real-time stock level monitoring
